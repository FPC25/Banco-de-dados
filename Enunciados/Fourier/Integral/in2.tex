Represente a função:
\begin{equation*}
    f(x) = \begin{cases}
    1 \hspace{0.3cm} se \hspace{0.3cm} 0 < x < a\\
    0 \hspace{0.3cm} se \hspace{0.3cm} x > a
    \end{cases}
\end{equation*}

como uma:
\begin{itemize}
    \item[\textbf{a)}] integral de Fourier cosseno;
    \item[\textbf{b)}] integral de Fourier seno.
\end{itemize}
