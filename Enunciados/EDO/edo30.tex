Ache a solução geral das seguintes E.D.O.'s lineares de 2ª ordem do tipo Euler-Cauchy:
\begin{itemize}
    \item[\textbf{a)}] $x^2y'' + xy' + y = 0$;
    \item[\textbf{b)}] $x^2y'' + 4xy' + 2y = 0$;
    \item[\textbf{c)}] $x^2y'' - 4xy' - 6y = 0$;
    \item[\textbf{d)}] $x^2y'' - 4xy' + 6y = 0$;
\end{itemize}
